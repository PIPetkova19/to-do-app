# To Do App
Приложение за управление на задачи, създадено с **Spring Boot**.

## Функции
-  CRUD
-  Филтриране на задачи реализирано чрез Strategy design pattern
-  Опция за споделени задачи
  
## Технологии
- Spring Boot – backend framework
- Spring Data JPA – data persistence
- Hibernate – ORM
- Microsoft SQL Server – database
- Maven – build tool
- MapStruct - mapping dtos & entities
- Liquibase - database schema change management
